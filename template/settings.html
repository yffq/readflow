API Usage

Save Article

POST /api/v1/save
Authorization: Bearer YOUR_API_KEY
Content-Type: application/json

{"url": "https://example.com/article", "html": "<p>Full article...</p>", "title": "Title"}

Export Articles

GET /api/v1/export?updated_after=2024-01-01T00:00:00Z&limit=100
Authorization: Bearer YOUR_API_KEY

Delete Articles

POST /api/v1/delete
Authorization: Bearer YOUR_API_KEY
Content-Type: application/json

{"ids": ["article-id-1", "article-id-2"]}
